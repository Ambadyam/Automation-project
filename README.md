#Project Goal

The goal of this project is to automate functional and UI testing of a web application to
1.Improve test coverage
2.Reduce manual testing effort
3.Ensure application reliability

#Key Contributions

1.Automated test scenarios using Selenium WebDriver.
2.Validated UI elements (buttons, links, logos) and page properties (title, URL).
3.Implemented data-driven testing to validate multiple input scenarios.
4.Verified HTTP response codes to ensure application reliability.
5.Logged and reported defects based on test execution results.

#Tool and Technologies

Programming Language: Java
Automation Tool: Selenium WebDriver
Testing Frameworks: TestNG, JUnit
Build & Dependency Management: Maven
Browser used: Chrome
