#Project Goal

The goal of this project is to automate functional and UI testing of a web application to:

Improve test coverage

Reduce manual testing effort

Ensure application reliability

Key Contributions

Automated test scenarios using Selenium WebDriver.

Validated UI elements (buttons, links, logos) and page properties (title, URL).

Implemented data-driven testing to validate multiple input scenarios.

Verified HTTP response codes to ensure application reliability.

Logged and reported defects based on test execution results.
